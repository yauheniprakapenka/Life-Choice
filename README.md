# Жизненный выбор

Оптимизировано для Swift 5 и Xcode 10.3

<b>Возможности:</b>
- Параллакс
- Логика переключателей
- Анимация
- Задержка отображения элементов по времени

<b>Не доработано:</b>
1. Адаптировано только для iPhone Xr
